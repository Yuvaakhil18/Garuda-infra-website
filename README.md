# 🏗️ Garuda Development Website

A modern and responsive corporate website developed for **Garuda Infra**, designed to showcase the company's services, projects, expertise, and contact information through a professional digital presence.

## 🚀 Overview

The Garuda Infra Website provides visitors with an easy-to-navigate platform to learn about the company, explore its services, view project information, and get in touch with the team.

## ✨ Features

- Responsive design for desktop, tablet, and mobile devices
- Modern and professional user interface
- Company introduction and profile section
- Services showcase
- Project highlights
- Contact information section
- Smooth navigation and user-friendly experience
- Fast-loading static website

## 🛠️ Technologies Used

- **HTML5** – Structure and content
- **CSS3** – Styling and responsive design
- **JavaScript** – Interactive functionality

## ⚙️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/Yuvaakhil18/Garuda-infra-website.git
```

### Open the Project

1. Navigate to the project folder.
2. Open `index.html` in your browser.

OR

Use VS Code Live Server:

```bash
Right Click → Open with Live Server
```
